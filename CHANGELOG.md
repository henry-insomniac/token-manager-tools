# 更新日志

## v0.1.0-preview.11

发布日期：2026-04-13

- 新增桌面客户端预览分发包：macOS 压缩包内提供 `Token Manager Tools.app`，Windows 压缩包内提供 `token-manager-desktop.exe`
- 新增应用图标资源和跨平台 release 打包脚本，分发包统一附带启动脚本、图标文件和 `SHA256SUMS.txt`
- 抽离 `internal/appservice` 作为共享业务层，Web 和桌面客户端改为复用同一套账号池服务
- 前端增加 transport 适配层，桌面客户端优先直连 Go 服务，浏览器模式继续走本地 HTTP API
- 桌面客户端登录继续使用系统浏览器，callback 端口会自动避让，避免和 `token-manager start/serve` 默认 `1455` 撞口
- 桌面客户端支持关闭后隐藏、单实例唤回和开机启动，窗口内增加快捷操作区，方便直接查看额度和执行刷新

## v0.1.0-preview.10

发布日期：2026-04-13

- 修复浏览器 OAuth 已登录但回调页出现 `ERR_EMPTY_RESPONSE` 的问题
- `serve/start` 启动的本地服务现在同时接收 `127.0.0.1` 和 `localhost` 回调，避免 loopback host 不一致导致回调丢失
- 浏览器入口、回调示例和分发包启动提示统一改成 `http://localhost:<端口>/`
- 如果回调端口已被其他程序占用，会直接给出明确提示，提醒改用其他端口启动

## v0.1.0-preview.9

发布日期：2026-04-13

- Web UI 登录增加“登录辅助”面板，可重开登录页、当前页打开或复制登录链接
- 增加 `POST /api/profiles/{name}/login/complete`，支持粘贴最终回调地址或 `code` 完成本机写入
- `/auth/callback` 登录完成后会自动把结果带回账号池页面并自动返回，减少手动刷新
- 补充 OpenAI 身份验证页 `unknown_error` 的页面内操作提示，方便切到系统浏览器继续登录

## v0.1.0-preview.8

发布日期：2026-04-11

- 收紧自动切换区的按钮尺寸，统一状态胶囊与操作按钮的视觉规格
- 压缩自动切换区文案，减少标题、说明、状态和事件里的重复表达
- 保持自动切换后台检查为 `90-180 秒` 随机区间

## v0.1.0-preview.7

发布日期：2026-04-11

- 新增自动兜底切换：当前运行账号不可用时，自动切到有额度的账号
- 自动切换支持最近事件记录，可在页面里查看上次检查、上次切换和判断结果
- 自动切换后台检查改为随机区间触发，避免固定节奏请求
- 当前后台轮询区间为 `90-180 秒`
- 增加自动切换 API 和回归测试，覆盖立即切换、最小切换间隔和服务侧触发
